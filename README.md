# 🏗️ СтройМаркет AI

**СтройМаркет AI** — это высоконагруженная платформа электронной коммерции для продажи строительных материалов, инструментов и сопутствующих товаров. 

Проект объединяет классический e-commerce с передовыми технологиями: основным интерфейсом служит **Telegram Mini App**, уведомления и первичное взаимодействие осуществляются через **Telegram-бота**, а помощь покупателям оказывает **интеллектуальный ИИ-консультант на базе RAG** (Retrieval-Augmented Generation).

---

## 🌟 Ключевые возможности

### 🛒 E-commerce и Бизнес-процессы
* **Полноценный интернет-магазин:** Каталог (12+ категорий), карточки товаров, умный поиск и фильтрация.
* **Корзина и Оформление:** Сохранение между сессиями, применение промокодов, автоматический расчет стоимости.
* **Складской учет:** Поддержка нескольких складов, резервирование, инвентаризация, история движений.
* **Платежи:** Интеграция с российскими провайдерами (ЮKassa, Т-Банк Эквайринг), обработка webhooks, возвраты.
* **Программа лояльности:** Бонусные баллы, накопительные скидки, персональные акции.

### 📱 Telegram Ecosystem
* **Telegram Mini App:** Основной UI для покупателя (каталог, корзина, чекаут, профиль, ИИ-чат).
* **Telegram Bot:** Уведомления о статусах заказа, напоминания о брошенной корзине, поддержка.

### 🤖 Искусственный интеллект (AI & ML)
* **RAG-Консультант:** ИИ-помощник, который отвечает на строительные вопросы, подбирает материалы (например, «рассчитай утеплитель для фасада 50м²») и рекомендует товары.
* **База знаний:** Векторное хранилище инструкций, СНиПов, описаний товаров и FAQ.
* **Рекомендательная система:** Персональные подборки на основе истории покупок, просмотров и контекста диалога с ИИ.

## 🏛 Архитектура и Технологии

Проект построен с упором на масштабируемость, поддерживаемость и строгое разделение ответственности.

### Архитектурные паттерны
* **Clean Architecture & DDD (Domain-Driven Design):** Код разделен на слои (Domain, Application, Infrastructure, Presentation). Бизнес-логика изолирована от фреймворков и базы данных.
* **Порты и Адаптеры (Hexagonal):** Инфраструктура (БД, Telegram API) реализует интерфейсы (порты), определенные в Домене.
* **CQRS (Command Query Responsibility Segregation):** Взаимодействие с системой через явные Команды (Commands) и Запросы (Queries).

### Технологический стек
* **Backend:** Python 3.11+, FastAPI (для Mini App API), Aiogram 3.x (для Telegram Bot).
* **Database:** PostgreSQL (основные данные, склад, заказы), SQLAlchemy 2.0 (async).
* **AI & RAG:** LangChain / LlamaIndex, Векторные БД (Qdrant/Pgvector), LLM API.
* **DevOps & Infrastructure:** Docker, Docker Compose, Nginx. *(В планах: Kubernetes, Kafka, Prometheus/Grafana).*

## 📂 Структура проекта (DDD)

Проект строго следует принципам Domain-Driven Design:

```text
app/
├── domain/             # 🧠 Ядро системы. Сущности, Value Objects, интерфейсы (порты). Не знает о БД и сети.
├── application/        # ⚙️ Сценарии использования (Use Cases). Команды и обработчики. Оркестрирует процессом.
├── infrastructure/     # 🔌 Детали реализации. SQLAlchemy модели, Telegram клиенты, внешние API.
├── presentation/       # 🚪 Точки входа. FastAPI роутеры (для Mini App) и хендлеры бота.
├── config.py           # Конфигурация приложения.
└── main.py             # Точка входа (запуск Бота и API в одном event loop).
```

## 🚀 Быстрый старт (Локальная разработка)

Для запуска проекта используется Docker и Docker Compose.

### 1. Клонирование и настройка окружения
```bash
git clone <your-repo-url>
cd building-shop-bot
```

### 2. Создание `.env` файла
Создайте файл `.env` в корне проекта и заполните его:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
SECRET_KEY=ваш_секретный_ключ
DATABASE_URL=postgresql+asyncpg://stroymarket_user:stroymarket_password@db:5432/stroymarket_db
```

### 3. Запуск инфраструктуры и приложения
```bash
docker-compose up --build 
npm run dev
ngrok http 3000 
```
После успешного запуска:
* **PostgreSQL** будет доступна на `localhost:5432`.
* **FastAPI (Mini App API)** будет доступно на `http://localhost:8000` (Docs: `http://localhost:8000/docs`).
* **Telegram Bot** начнет опрос (polling) и ответит на команду `/start`.
